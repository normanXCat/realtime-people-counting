# realtime-people-counting
A real-time people counting pipeline built with YOLOv8 for detection and ByteTrack for multi-object tracking. Features virtual line crossing, entry/exit counter, and a live dashboard. STIC Master's thesis — 2026.

## Structure du projet

* **`docs/`** : Documents de recherche, étude bibliographique et planification du projet (Gantt).
* **`models/`** : Fichiers liés au modèle de détection YOLO (poids, configuration et classes).
